# NexusAI — Premium Full-Stack AI Platform

NexusAI is a high-performance, mobile-responsive AI orchestration platform built with a hybrid Node.js and Python architecture.

## 🚀 Features
- **Intelligent Chat**: Multi-modal chat with support for Groq, Llama 3, and more.
- **Strategic Planner**: Autonomous task decomposition and planning.
- **Knowledge Bases (RAG)**: Upload documents and query your own knowledge.
- **Data Science Engine**: AutoML, EDA, and data cleaning.
- **Code Workspace**: Built-in editor for Python/JS development.
- **Voice Mode**: Real-time voice transcription and interaction.

## 🛠️ Tech Stack
- **Frontend**: React, Tailwind CSS, Lucide Icons.
- **Gateway**: Node.js, Express, Socket.io.
- **Microservice**: Python, Flask, Groq.
- **Database**: MongoDB Atlas.

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/R123gh/Nexus-AI-.git

# Install all dependencies (Frontend, Server, Backend)
npm run install:all
```
